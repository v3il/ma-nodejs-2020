### Launch locally
1. Configure client project: \
```cd client```\
```npm ci```\
```npm run build```
2. Configure server project: \
```cd ..``` \
```npm ci```
3. Rename **.env.copy** to **.env** and fill it with your connection params
4. Run migrations: \
```npm run migrate```
5. Start server: \
```npm start```
6. Open http://localhost:XXXX/
